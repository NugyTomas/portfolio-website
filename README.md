# Portfolio Website

A personal portfolio website built with Flask to showcase my projects, skills, and developer journey.

🌐 **Live Demo:** https://tomasnguyen.com

---

## Features

- Responsive design
- Dynamic project cards loaded from a SQLite database
- Contact form with email integration
- Reusable Jinja2 templates
- Modern UI built with Bootstrap
- Easy to expand with new projects and skills

## Technologies

- Python 3
- Flask
- Jinja2
- Bootstrap 5
- HTML5
- CSS3
- SQLite
- SQLAlchemy

## Run Locally

Clone the repository:

```bash
git clone https://github.com/NugyTomas/portfolio-website.git
```

Install dependencies:

```bash
pip install -r requirements.txt
```

Create a `.env` file:

```env
MY_EMAIL=your_email@example.com
PASSWORD=your_app_password
```

Run the application:

```bash
python main.py
```

Then open:

```
http://127.0.0.1:5000
```
